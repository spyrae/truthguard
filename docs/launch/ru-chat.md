# Пост для чата (русский)

---

Сделал штуку, которая ловит AI-агентов на вранье.

Проблема: Claude Code (и другие AI-агенты) систематически врут о результатах. Говорит "тесты прошли" - а тесты не запускались. Говорит "файл обновил" - а файл не изменился. Пушит с --force без спроса. Это задокументированный баг, промпты не помогают - агент их просто игнорирует.

Решение: TruthGuard - набор хуков, которые проверяют заявления агента программно, на уровне tool calls. Не текстовые инструкции, а реальная верификация.

Что делает:
- Сравнивает SHA256 файла до/после редактирования - если не изменился, блокирует
- Проверяет exit code команд - если тесты упали, агент не может заявить что прошли
- Автоматически запускает тесты перед каждым git commit
- Блокирует --force, --no-verify, rm -rf
- После коммита заставляет агента подтвердить что фикс реально работает

За 2 дня на продакшн-проекте: 5 коммитов заблокировано (падающие тесты), 3 опасные команды заблокированы, 0 ложных срабатываний.

Установка за 30 секунд:
```
npx truthguard install && npx truthguard init
```

Работает с Claude Code и Gemini CLI. Чистый bash + jq, без бэкенда.

GitHub: https://github.com/spyrae/truthguard
npm: https://www.npmjs.com/package/truthguard
